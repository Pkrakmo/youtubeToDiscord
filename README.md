# youtubeToDiscord
Script that will get the latest youtube video from a spesific channel and post it to Discord via their Webhook

.ENV variables:

WEBHOOK=[URL]

WEBHOOKDEBUG=[URL]

WEBHOOKDEBUG is optional, but required if const debugMode: boolean = true
